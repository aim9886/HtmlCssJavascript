Dance School Registration Form

The "Dance School Registration Form" project is a React.js and Redux-based web application that uses MySQL for data storage. It features a comprehensive student registration form with personal and dance-related details. The system includes a View Manager for easy navigation and oversight of student registrations. The Academy Manager allows administrators to update general school information. A Course Manager facilitates the addition, editing, and deletion of dance courses. The project ensures data security by implementing user authentication. MySQL is employed for reliable data persistence. Responsive design principles guarantee a seamless experience on various devices. The application is designed for efficient management of dance school registrations, offering administrators tools for overseeing students, courses, and overall academy details. The integration of React.js, Redux, and MySQL provides a scalable and user-friendly solution for dance school administration.
